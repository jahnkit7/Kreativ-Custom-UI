Ce plugin ajoute automatiquement une ligne de personnalisation dans le fichier `nbdesigner-frontend-modern.php` et applique un style personnalisé au bouton "Finaliser" de la page de design "create-your-own".

== Fonctionnalités ==
* Ajout automatique du fichier custom-ui.php dans le fichier frontend du plugin NBDESIGNER
* Stylisation automatique du bouton de finalisation

== Installation ==
1. Uploadez le plugin dans `/wp-content/plugins/`
2. Activez-le via le menu 'Extensions' de WordPress
3. Rendez-vous sur la page "create-your-own" pour voir les effets
